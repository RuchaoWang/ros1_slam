# 提出了一种新的安全飞行走廊生成算法

## 思路介绍
1. 结合了安全飞行走廊和欧几里德距离场
2. 目前已出现的安全飞行走廊是基于离散空间的走廊扩展方法，本方法采用连续搜索方法
3. 前端采用A*算法，连续是根据路径点求解法向量，类似包络线的扩展方法，求解向量的方法采用点与点之间形成的向量，然后求解法向量
4. 生成安全走廊后，根据相交扩展区域和连续区域采样求出优化路径点
5. 在连续区域和相交扩展区域根据到上一个点的距离代价、到终点的代价、重合代价、以及最重要的欧几里德距离场代价求解出最优解
6. 最终使用贝塞尔曲线进行优化插值

## 连续走廊生成方法
原本是想通过求解原本路径点切线的方式，再根据切线构造法向量，通过设定搜索长度，生成连续安全飞行走廊，但是求解导数或者说切线斜率面临两个重要的问题，当k=0和k=无穷的时候，会出现重大问题，所以经过考虑决定采用三角形的方法
